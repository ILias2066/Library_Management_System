# Library Management System

Это простое приложение для управления библиотекой, написанное на Python. Оно включает функциональность для добавления, удаления, поиска и отображения книг, а также для их выдачи и возврата пользователями.

## Описание проекта

Приложение позволяет:

- Добавлять новые книги в библиотеку.
- Удалять книги из библиотеки.
- Отображать список всех книг в библиотеке.
- Пользователи могут брать книги и возвращать их.
- Каждая книга имеет статус доступности.

## Структура проекта

- **`book.py`**: класс для представления книги с аттрибутами и методами для выдачи и возврата.
- **`library.py`**: класс для управления библиотекой, включая добавление, удаление и поиск книг.
- **`user.py`**: класс для представления пользователя, который может брать и возвращать книги.

## Запуск проекта

1. **Клонируйте репозиторий**:
    ```bash
    git clone https://github.com/yourusername/library-management.git
    cd library-management
    ```

2. **Создайте и активируйте виртуальное окружение** (опционально):
    ```bash
    python -m venv venv
    source venv/bin/activate  # Для Linux/macOS
    venv\Scripts\activate     # Для Windows
    ```

3. **Установите зависимости**:
    В проекте может быть файл `requirements.txt`, в котором указаны необходимые зависимости. Чтобы установить их, выполните:
    ```bash
    pip install -r requirements.txt
    ```

4. **Запустите приложение**:
    Вы можете запустить ваше приложение с помощью следующей команды:
    ```bash
    python main.py
    ```

## Структура Docker

### Dockerfile

В проекте есть файл **`Dockerfile`**, который используется для контейнеризации приложения. Чтобы создать Docker-образ и запустить приложение в контейнере, выполните следующие шаги:

1. **Соберите Docker-образ**:
    ```bash
    docker build -t library-app .
    ```

2. **Запустите контейнер**:
    ```bash
    docker run -p 8000:8000 library-app
    ```

3. **Использование Docker Compose**:
    Если вы используете Docker Compose для упрощения работы с несколькими контейнерами, выполните:
    ```bash
    docker-compose up --build
    ```

## Функции

### Классы:

- **Book**: Представляет книгу с атрибутами `id`, `title`, `author`, `year`, `available` и методами для управления статусом книги.
- **Library**: Управляет коллекцией книг в библиотеке (добавление, удаление, отображение).
- **User**: Представляет пользователя, который может брать книги и возвращать их в библиотеку.

### Пример использования:

```python
# Создаем книги
book1 = Book(1, "1984", "George Orwell", 1949)
book2 = Book(2, "To Kill a Mockingbird", "Harper Lee", 1960)

# Создаем библиотеку и добавляем книги
library = Library()
library.add_book(book1)
library.add_book(book2)

# Создаем пользователя
user = User("Иван")

# Пользователь берет книгу
print(user.borrow_book(book1, library))

# Пользователь пытается вернуть книгу
print(user.return_book(book1, library))

# Отображаем все книги в библиотеке
print(library.list_books())
